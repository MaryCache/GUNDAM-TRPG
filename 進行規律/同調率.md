# 同調率システム（Synchronization Rate）

## 概要

同調率とは、パイロットと機体の精神的同期度を表すパラメータです。  
戦闘中に特定のARTSや状況によって変動し、機体の性能に直接影響を与えます。

---

## 基本仕様

```
初期値: 0
範囲: -6 ～ +6 (ランク制)
リセット: 戦闘終了時に0に戻る
```

---

## 同調率ランクと倍率

同調率は **ポケモンの能力ランク補正** と同じ計算式を使用します。

| ランク | 倍率 | 分数表記 | 効果 |
|:---:|:---:|:---:|:---|
| **+6** | 400% | 8/2 | 圧倒的優位。全ての判定が極めて有利 |
| **+5** | 350% | 7/2 | 大幅な性能向上。至高の領域 |
| **+4** | 300% | 6/2 | 著しい性能向上 |
| **+3** | 250% | 5/2 | 明確な優位性 |
| **+2** | 200% | 4/2 | 大きなアドバンテージ |
| **+1** | 150% | 3/2 | わずかな優位性 |
| **0** | 100% | 2/2 | 通常状態（補正なし） |
| **-1** | 67% | 2/3 | わずかな性能低下 |
| **-2** | 50% | 2/4 | 明確な性能低下 |
| **-3** | 40% | 2/5 | 著しい性能低下 |
| **-4** | 33% | 2/6 | 大幅な性能低下 |
| **-5** | 29% | 2/7 | 機体制御が困難 |
| **-6** | 25% | 2/8 | 最悪の同期不全。パイロットと機体が反発 |

---

## 同調率の影響範囲

同調率は戦闘計算の**最終段階**で精度指数・回避指数に倍率補正をかけます。

### 1. 精度指数への影響

**正しい計算順序**:
```
① 精度率 = 基礎精度率 - RANGE減衰 + RANGE補正
② 精度指数(Raw) = 技能行使Lv × 精度率
③ 最終精度指数 = 精度指数(Raw) × 同調率倍率
```

**例**:
- 技能行使Lv5、精度率120%の場合
  - 精度指数(Raw) = 5 × 1.2 = 6.0
  - 同調率+2 (200%倍率) → **最終精度指数 = 6.0 × 2.0 = 12.0**
  - 同調率-2 (50%倍率) → **最終精度指数 = 6.0 × 0.5 = 3.0**

### 2. 回避指数への影響

**正しい計算順序**:
```
① 回避率 = 100% + [10% × (機動 - 精度指数)]
② 回避指数(Raw) = 機動制御Lv × 回避率
③ 最終回避指数 = 回避指数(Raw) × 同調率倍率
```

**例**:
- 機動制御Lv6、回避率150%の場合
  - 回避指数(Raw) = 6 × 1.5 = 9.0
  - 同調率+1 (150%倍率) → **最終回避指数 = 9.0 × 1.5 = 13.5**
  - 同調率-3 (40%倍率) → **最終回避指数 = 9.0 × 0.4 = 3.6**

## 同調率の変動

### 上昇方法

1. **CORE:ARTS『OVER:DOSE』**: 同調率/永続上昇: +2
2. **CORE:ARTS『MODE:ULT』**: 同調率/強化: +2 (このTURNのみ)
3. **特定のCUSTOM:ARTS効果**

### 下降方法

1. **CORE:ARTS『SYNK:BLOCK』**: 敵機の同調率/永続下降: -2
2. **部位破壊ペナルティ** (オプションルール)
3. **精神的ショック** (仲間の撃墜など、GMの裁量)

---

## 用語の違い

### 「強化」vs「永続上昇」vs「上昇」

| 用語 | 効果期間 | 例 |
|:---|:---|:---|
| **強化** | このTURNのみ有効 | MODE:ULTの「同調率/強化:+2」 |
| **永続上昇** | 戦闘終了まで有効 | OVER:DOSEの「同調率/永続上昇:+2」 |
| **上昇** | 戦闘終了まで有効（永続と同義） | CUSTOM:ARTSで使用される一般表記 |

**重要**: 「強化」は一時的、「永続上昇/上昇」は戦闘中持続します。

---

## 戦術的運用

### 攻撃型運用
- OVER:DOSEで同調率+2を獲得し、攻撃精度を大幅に向上
- ただし被弾時に効果喪失のリスクあり

### 妨害型運用
- SYNK:BLOCKで敵の同調率を-2にし、相手の性能を低下
- 遠距離から妨害に専念する戦術

### 切り札型運用
- MODE:ULTで一時的に同調率+2を得て、必殺の一撃を放つ
- 戦闘中1度のみの使用制限に注意

---

## 設計意図

同調率システムは、従来のTRPGにおける「単純なバフ/デバフ」を、**段階的なリスク管理**に昇華させたものです。

- **+6まで積み重ねる戦略**: 長期戦での優位性確保
- **マイナスからの回復戦術**: 妨害を受けた際の立て直し
- **範囲制限(-6～+6)**: 無限の積み上げによるバランス崩壊の防止

パイロットと機体の「絆」を数値化し、戦闘における駆け引きの軸として機能します。

## 計算上の注意

同調率による倍率補正のタイミングは以下の通りです:

```
戦闘計算の流れ:
1. 精度率計算（基礎精度率 - RANGE減衰 + RANGE補正）
2. 精度指数(Raw)計算（技能行使Lv × 精度率）
3. ★ 同調率倍率を精度指数に適用 → 最終精度指数
4. 回避率計算（100% + [10% × (機動 - 精度指数)]）
5. 回避指数(Raw)計算（機動制御Lv × 回避率）
6. ★ 同調率倍率を回避指数に適用 → 最終回避指数
7. 命中指数計算（最終精度指数 - 最終回避指数）
8. ダメージ計算（命中指数 - 装甲）
```

**重要**: 同調率補正は、精度指数・回避指数の計算後、命中指数計算の前に適用されます。

小数点処理: 同調率補正後も **小数点第二位まで保持**（第三位を四捨五入）。
