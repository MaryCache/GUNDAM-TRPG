## 🎲 技能判定

### ダイスロール
```
1d100を振って、出目で技能Lvが変動

01-05: +5Lv (革命的成功)
06-23: +2Lv
24-41: +1Lv
42-59: ±0Lv
60-77: -1Lv
78-95: -2Lv
96-00: -5Lv (致命的失敗)
```

### 技能評価
```
Lv.0以下: 致命的失敗
Lv.1-2: 未熟
Lv.3-4: 平凡
Lv.5-6: 優良
Lv.7-8: 卓越
Lv.9-10: 至高
Lv.11以上: 革命的成功
```

## 🎯 タスク難度

```
Lv.1: 平易
Lv.2: 単純
Lv.3: 基礎
Lv.4: 応用
Lv.5: 複雑
Lv.6: 困難
Lv.7: 過酷
Lv.8: 鬼門
Lv.9: 理外
Lv.10: 不可
```

**判定**:
- 技能行使Lv ≧ タスクLv → 成功
- 技能行使Lv < タスクLv → 失敗

## 🔗 同調率システム

パイロットと機体の精神的同期度を表すパラメータ。  
**-6 ～ +6** のランクで管理され、精度指数・回避指数に倍率補正がかかります。

### 倍率テーブル(抜粋)
```
+6: 400% (圧倒的優位)
+3: 250% (明確な優位)
 0: 100% (通常)
-3:  40% (著しい性能低下)
-6:  25% (最悪の同期不全)
```

詳細: [同調率.md](同調率.md)
