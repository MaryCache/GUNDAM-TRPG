# クイックスタートガイド

GUNDAM-TRPGを今すぐ始めたい方向けの5分ガイドです。

---

## 🎯 このゲームとは?

**GUNDAM-TRPG**は、ガンダム世界観のロボット戦闘をシミュレートするTRPGシステムです。
- **Web・Botを連携**したデジタル運用システム
- **駆け引きと戦略性**を重視した戦闘システム
- **ユーザー創作**を前提とした機体・武器設計

---

## 🚀 3ステップで始める

### ステップ1: キャラクターを作る

[軍籍付与申請フォーム](https://my-character-sheet.vercel.app/)にアクセス

1. **能力値を配分** (21点を6つの能力に振り分け)
2. **技能を強化** (80点で好きな技能をレベルアップ)
3. **JSON出力** してDiscord Botに登録

**初心者におすすめの能力配分**:
```
身体:3, 精神:3, 器用:5, 知性:4, 五感:5, 外見:1
→ 器用と五感を高めてパイロット向き
```

詳細: [従軍者作成規律](作成規律/従軍者作成規律.md)

---

### ステップ2: 機体を作る

[A.R.S 機体設計申請フォーム](https://ms-builder.vercel.app/)にアクセス

1. **TLvを決める** (GMが指定、通常は10)
2. **Type選択** (F:近接/S:遠距離/E:バランス)
3. **機動と積載を配分** (合計=TLv)
4. **武器とARTSを設計**
5. **JSON出力** してDiscord Botに登録

**初心者におすすめの機体**:
```
Type:E (バランス型), TLv:10
機動:6, 装甲:2, 積載:4
武器:ビームライフル(重量2)
→ 扱いやすい標準機
```

詳細: [MS作成規律](作成規律/MS作成規律.md)

---

### ステップ3: Discordで遊ぶ

Discord Botに参加してGMの指示に従う

**基本コマンド**:
```
/pc import <JSON>    - キャラクター登録
/ms import <JSON>    - 機体登録
/board create        - 戦闘盤面作成
/acc                 - 精度指数計算
/avoid               - 回避指数計算
```

## 📖 基本ルール早見表

### 技能判定
```
1d100を振って、出目で技能Lvが変動
01-05: +5Lv (クリティカル)
06-23: +2Lv
24-41: +1Lv
42-59: ±0Lv
60-77: -1Lv
78-95: -2Lv
96-00: -5Lv (ファンブル)
```

### 戦闘の流れ
```
1. SELECT PHASE  - 全員が行動を秘密裏に予約
2. ACTION PHASE  - 速度順に行動を実行
3. END PHASE     - ターン終了、次のWAVEへ
```

### ダメージ計算
```
精度指数 = 技能行使Lv × 精度率 × 同調率倍率
回避指数 = 機動制御Lv × 回避率 × 同調率倍率
命中指数 = 精度指数 - 回避指数
発生ダメージ = 命中指数 - 装甲 (1ダメージ = 部位1つ破壊)
```

## 🆘 よくある質問

### Q1. TLvはいくつにすればいい?
**A**: GMが指定しますが、基準は**TLv.10**です。初心者はこれで始めましょう。

### Q2. Typeはどれを選べばいい?
**A**: 初めてなら**Type:E**(バランス型)が無難です。慣れたらF(近接特化)やS(遠距離特化)も試してみましょう。

### Q3. 技能点80点をどう使えばいい?
**A**: 主力技能(機動制御・近接剣術・精密射撃のいずれか)に**+3(30点)**、サブ技能に残りを振るのがおすすめ。

### Q4. 同調率はどうすれば上がる?
**A**: CORE:ARTS『OVER:DOSE』(+2)や『MODE:ULT』(+2)で上昇します。敵の『SYNK:BLOCK』(-2)で下がることも。

### Q5. 装甲0になったらどうなる?
**A**: 次の被弾で部位が直接破壊されます。【コックピット】破壊=絶命なので要注意!

## 🎮 遊び方のヒント

### 初心者向け戦術
1. **距離を保つ** - 遠距離から射撃して相手の接近を許さない
2. **SHIFT:SAFEを活用** - 精度指数0にできる防御ARTS(優先度+3)
3. **MODE:ULTは温存** - 戦闘中1度のみの切り札。ここぞという時に

### 中級者向け戦術
1. **OVER:DOSEで先制** - 同調率+2で精度・回避を大幅強化
2. **OVER:RIDEで柔軟に** - 相手の動きを見てから行動変更(難易度高)
3. **部位狙い** - 4ダメージ以上なら狙った部位を破壊可能
