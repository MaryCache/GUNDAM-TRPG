## 機体設計チェックリスト

### 基本制約の確認

- [ ] **機動 + 積載 = TLv** が成立しているか?
- [ ] **積載 = WEAPON総重量 + 装甲** が成立しているか?
- [ ] **ARTS総数 ≦ TLv** になっているか? (CORE:ARTS 4つ + CUSTOM:ARTS、上限=TLv)
- [ ] **Type** は F / S / E のいずれかか?

### TLv別の妥当性チェック

#### TLv.8 (低性能機)
```
機動: 4~6
装甲: 1~3
積載: 2~4
WEAPON総重量: 1~3
ARTS総数: 8個
```

#### TLv.10 (標準機)
```
機動: 5~7
装甲: 2~4
積載: 3~5
WEAPON総重量: 2~4
ARTS総数: 10個
```

#### TLv.12 (高性能機)
```
機動: 6~9
装甲: 2~5
積載: 3~6
WEAPON総重量: 2~5
ARTS総数: 12個
```

### よくある設計ミス

❌ **機動10, 装甲0, 積載0** (TLv.10)
→ 積載0ではWEAPONを1つも装備できない!

❌ **機動3, 装甲5, 積載7** (TLv.10)
→ 機動が低すぎて行動順が最後になる

✅ **機動6, 装甲2, 積載4** (TLv.10)
→ バランスが取れている!

## WEAPON設計チェックリスト

### TLv.10のWEAPON基準

- [ ] **故障値**: 75~90 の範囲か?
- [ ] **装填数**: 装填ありなら3~6、装填なしなら null
- [ ] **重量**: 1~3 の範囲か?
- [ ] **使用部位**: 頭部/左腕/右腕/左脚/右脚/左翼/右翼/コックピット のいずれかか?

### 故障値と装填数のバランス

**計算式**: `TLv ≒ (故障値 - 90) + (装填数 × 0.5)`

#### TLv.10 の例
```
故障値85, 装填6 の場合:
(85 - 90) + (6 × 0.5) = -5 + 3 = -2 ≒ TLv.10の下限

故障値80, 装填8 の場合:
(80 - 90) + (8 × 0.5) = -10 + 4 = -6 ≒ TLv.10の標準

故障値90, 装填4 の場合:
(90 - 90) + (4 × 0.5) = 0 + 2 = 2 ≒ TLv.12相当
```

### 装填なしWEAPONの場合

```
TLv.10の場合
基準故障値: 80~90

TLv.8の場合
基準故障値: 75~85
```

### よくある設計ミス

❌ **故障値95, 装填10, 重量1** (TLv.10)
→ 故障しにくく、弾数も多すぎる!オーバースペック

❌ **故障値70, 装填2, 重量5** (TLv.10)
→ 故障しやすく、弾数少なく、重すぎる!

✅ **故障値85, 装填6, 重量2** (TLv.10)
→ 標準的なビームライフル、バランス良し!

## ARTS設計チェックリスト

### 基本要素の確認

- [ ] **ARTS.No.**: 5以降の整数か? (1~4はCORE:ARTS専用)
- [ ] **依存武器**: 装備しているWEAPONか?
- [ ] **依存技能**: 操縦技能(機動制御/近接剣術/精密射撃/危機感知)のいずれかか?
- [ ] **有効射程**: 1~10 程度の妥当な値か?
- [ ] **基礎精度率**: 40~80% の範囲か?

### TLv別の基礎精度率目安

| TLv | 基礎精度率 | 特殊効果段階 |
|:---:|:---:|:---|
| 5~7 | 40~55% | 弱い効果(1~2つ) |
| 8~10 | 50~65% | 標準効果(2~3つ) |
| 11~14 | 60~75% | 強い効果(3~4つ) |
| 15~20 | 70~85% | とても強い効果(4~5つ) |

### 特殊効果のバランス

**弱い効果の例**:
- ダメージ+1
- 精度率+10%
- 次のWAVE中、回避率+10%

**標準効果の例**:
- ダメージ1.5倍
- 部位選択権を獲得
- 同調率+1(永続)

**強い効果の例**:
- ダメージ2倍
- 相手の行動権を1消費させる
- 同調率+2(永続)

### 優先度の扱い

```
優先度なし: 通常ARTS (速度指数で行動順決定)
優先度+1: 先制ARTS (速度指数無視)
優先度+2: 超先制ARTS (MODE:ULTなど)
優先度+3: 最優先ARTS (SHIFT:SAFEなど)
```

**注意**: 優先度+1以上のARTSは **OVER:RIDE不可** にすることを推奨

### よくある設計ミス

❌ **基礎精度率100%, 有効射程10, ダメージ2倍**
→ 強すぎる!当たりやすく、射程も長く、威力も高い

❌ **基礎精度率30%, 有効射程1, 特殊効果なし**
→ 弱すぎる!当たらず、射程も短い

✅ **基礎精度率60%, 有効射程5, 命中時ダメージ1.5倍**
→ TLv.10標準ARTS、バランス良し!

## 禁止事項

以下の効果を持つARTSは**作成禁止**です。ゲームバランスを著しく損ないます。

### ❌ 絶対禁止

#### 1. 無限リソース系
```
❌ 装甲を無限回復する効果
❌ 故障したWEAPONを戦闘中に修復
❌ 破壊された部位を復元
❌ 無限に同調率を上昇させる
```

#### 2. 一方的な優位性
```
❌ 相手の精度指数を0にする (SHIFT:SAFEは例外)
❌ 相手の回避指数を0にする
❌ 相手の機動を0にする
❌ 必中効果 (精度指数9999など)
```

#### 3. 複数ターン行動系
```
❌ 1TURNに複数回行動できる
❌ 相手の行動権を2つ以上奪う
❌ 相手のTURNをスキップさせる
```

#### 4. 過剰な同調率操作
```
❌ 1WAVE中に同調率を+5以上変動
❌ 相手の同調率を-4以上変動
❌ 同調率の上下限(-6~+6)を超える
```

#### 5. ダメージ上限突破
```
❌ 一撃で部位を5つ以上破壊
❌ 【コックピット】を指定破壊 (4ダメージでランダムの可能性はOK)
❌ 全部位を同時破壊
```

### ⚠️ 条件付きで許可

以下は**GMの許可があれば**使用可能です:

#### 1. 高リスク・高リターン
```
⚠️ 1WAVE中、同調率+4だが被弾時に-6になる
⚠️ 精度率+50%だが、機動が半分になる
⚠️ ダメージ3倍だが、自分も2ダメージ受ける
```

#### 2. 限定的な強効果
```
⚠️ 戦闘中1度のみ、部位を1つ回復
⚠️ 戦闘中1度のみ、装甲を全回復
⚠️ 戦闘中1度のみ、故障したWEAPONを修復
```

#### 3. 特殊な発動条件
```
⚠️ 装甲0の時のみ発動可能、必中+ダメージ2倍
⚠️ 同調率-4以下の時のみ、同調率を0にリセット
⚠️ 部位が3つ以下の時のみ、全回復
```

## バランス調整のヒント

### 強すぎるARTSの調整方法

#### 方法1: デメリットを追加
```
変更前: 精度率+50% (強すぎる)
変更後: 精度率+50%、ただし機動-3 (リスク追加)
```

#### 方法2: 使用回数制限
```
変更前: ダメージ2倍 (連発されると強すぎる)
変更後: ダメージ2倍、戦闘中2回まで (装填制限)
```

#### 方法3: 発動条件を厳しく
```
変更前: 同調率+3 (簡単に使える)
変更後: 同調率+3、ただし優先度+1で被弾時に効果消失
```

### 弱すぎるARTSの調整方法

#### 方法1: 精度率を上げる
```
変更前: 基礎精度率40% (当たらない)
変更後: 基礎精度率60% (標準的)
```

#### 方法2: 特殊効果を追加
```
変更前: 通常攻撃のみ (地味)
変更後: 命中時、相手の同調率-1 (デバフ追加)
```

#### 方法3: 射程を延ばす
```
変更前: 有効射程1 (近接のみ)
変更後: 有効射程3 (中距離対応)
```

## ✅ 設計完了チェックリスト

### 機体設計完了時
- [ ] TLv計算式が成立している
- [ ] Type恩恵効果を記載している
- [ ] CORE:ARTS 4つを含めてARTS総数≦TLv (上限を超えていない)
- [ ] 全WEAPONの重量合計が積載以下
- [ ] 装甲値が妥当な範囲

### WEAPON設計完了時
- [ ] 故障値と装填数のバランスが取れている
- [ ] 重量がTLvに見合っている
- [ ] 使用部位が明確

### ARTS設計完了時
- [ ] 依存武器・依存技能が明確
- [ ] 基礎精度率がTLvに見合っている
- [ ] 特殊効果が禁止事項に抵触していない
- [ ] 優先度ありARTSはOVER:RIDE不可を記載

## 🎓 まとめ

**良いデザインの鍵**:
1. 💪 **リスクとリターンのバランス**
2. 🎯 **明確な役割と特徴**
3. ⚖️ **相手にも対抗手段がある**

**避けるべきデザイン**:
1. ❌ **一方的に有利な効果**
2. ❌ **リスクのない強力な効果**
3. ❌ **相手が何もできなくなる効果**

独自の機体・武器を作る楽しみを、ぜひ味わってください!🎨✨
