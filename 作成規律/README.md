## 🚀 作成の流れ

### 1. キャラクター作成
```
① 能力値を配分 (21点)
② 技能を強化 (80点)
③ JSON出力
```
→ 詳細: [従軍者作成規律](従軍者作成規律.md) / [軍籍付与申請フォーム](https://my-character-sheet.vercel.app/)

### 2. 機体作成
```
① TLvを決める (GMが指定)
② Typeを選ぶ (F/S/E)
③ 機動・装甲・積載を配分
④ WEAPON・ARTSを設計
⑤ JSON出力
```
→ 詳細: [MS作成規律](MS作成規律.md) / [A.R.S 機体設計申請フォーム](https://ms-builder.vercel.app/)

### 3. バランス確認
```
① チェックリストで確認
② 禁止事項に抵触していないか確認
③ GMに提出
```
→ 詳細: [バランスガイドライン](バランスガイドライン.md)
