```
『SHIFT：SAFE』
ARTS.No.：3
依存武器：null
依存技能：null
有効射程：null
基礎精度率：null
─=≡効果≡=─
①：このARTSの依存武器の使用部位は【頭部】として扱う。
②：優先度：+3
③：このARTSを使用したWAVE中、自身に対して発生した精度指数を0にする。
④：③の効果が発動した場合、自身の装甲を1消費する。
⑤：このARTSの依存武器の使用部位が破壊されているとき、従軍者の操縦技能【危機感知】がLv8以上であれば以下の効果を発動する。
使用部位ルールを無視して、このARTSを使用してもよい。
⑥：このARTSは【OVER：RIDE】で指定できない。
─=≡詳細≡=─
全ての従軍者が初歩として学ぶ防御姿勢。
メインカメラ、センサーによる戦況情報を取得した上で反射運動を駆使し、回避と盾受けを同時に成立させる。
ただし被弾時には装甲を損耗する、損害を前提とした行動である点には注意が必要。
この姿勢を取るか否かの判断は、戦場における胆力と直感を試す。
```
```
『MODE：ULT』
ARTS.No.：4
依存武器：null
依存技能：null
有効射程：null
基礎精度率：null
─=≡効果≡=─

このTURN中、発動したARTSの射程距離を自由に減少させてもよい。
⑤：このアーツは戦闘中、一度しか発動できない。
⑥：このアーツは【OVER：RIDE】で指定できない。

機体のリアクターを臨界近くまで押し上げ、出力制御を強制解除する最終戦闘モード。  
この状態では全ての安全装置が停止し、神経接続は過熱域に突入する。  
一瞬の爆発的出力を得る代償として、戦闘後の機体・精神への負荷は計り知れない。
```
