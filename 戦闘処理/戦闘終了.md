## 離脱/妨害
```
戦闘参加機体は、任意で『離脱』を試みることができる。
また、いずれかの理由で使用できるARTSがなくなったとき、従軍者は戦闘から『離脱』しなくてはならない。
戦闘参加機体は、『離脱』を試みる機体を『妨害』することが出来る。
『妨害』する場合、両機体の従軍者は【機動制御】の技能を使用する。
『離脱』側の技能行使Lv.≧『妨害』側の技能行使Lv.のとき、『離脱』は成功し、『離脱』した機体は戦闘を終了する。
『離脱』側の技能行使Lv.<『妨害』側の技能行使Lv.のとき、『離脱』は失敗し、戦闘を継続する。
```
## 絶命
```
部位【コックピット】を破壊されたとき、その機体の従軍者は『絶命』する。
『絶命』した従軍者は以降のシナリオで使用することが出来ない。
```
