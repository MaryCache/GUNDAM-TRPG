## 『戦闘区域作成』
```
戦闘開始の請求はシナリオ進行中にGMより行われる。
PLあるいはPCの能動的行動によって戦闘が発生する場合でも、GMによって戦闘開始の請求が行われたと解釈する。
戦闘区域を作成するためには、【機械操作】の技能によって該当区域をジャミングし、はるか遠方からの索敵やレーザー照射を防がねばならない。
作成された戦闘区域の座標数は、成功した【機械操作】の技能Lv+5以下の数から指定する。指定は技能成功者が行うものとする。
```
```
/board create：座標盤面作成。GMが行う。
/board add：座標盤面にキャラクターを追加。それぞれのPLが行う。
```
