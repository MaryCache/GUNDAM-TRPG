### ❶ARTS使用宣言
```
②TURN ACTIONを行う機体は、行動権を1消費して、1.SELECT PHASEにて予約したARTSの使用を宣言する。
予約ARTS/予約対象を変更して使用する場合、【OVER：RIDE】の条件を満たして行動権を1消費する必要がある。

(予約ARTSの有効射程)-(予約対象と自身の座標差量)≧0 のとき：❷に移行する。

(予約ARTSの有効射程)-(予約対象と自身の座標差量)<0 のとき：予約対象をARTSの対象とすることは出来ない。
このとき座標上に他機体が存在し、そのうちのいずれかがこのARTSの有効射程内に存在する場合、その中から一機を選び、【OVER：RIDE】を使用せず予約対象を上書きできる。選択した対象に対して、❷に移行する。
このとき有効射程内に対象とできる機体が存在しない場合、ARTSは不発扱いとなる。
ARTSを不発したとき、行動権を消費して❼に移行する。

※予約対象が撃破・撤退等で戦闘参加機体から除外されている場合も、上記と同様に対象再指定の手順を用いる。
　
座標差量の定義について
(AとBの座標差量)＝(Aの座標値)-(Bの座標値)
このとき、(座標差量)は絶対値(正負の符号を参照しない)として扱う。
```
### ❷故障判定
```
ARTS使用機体は、ARTSの依存技能で判定する。
(判定値)≧(使用ARTSの依存武器の故障値)のとき、そのWEAPONを破壊し、ARTSの処理を行わずに❻に移行する。
(判定値)<(使用ARTSの依存武器の故障値)のとき、❸に移行する。
```
### ❸ARTS効果発生
```
記載されたARTSの効果の処理を開始する。
ARTSによる負荷がARTS対象機体に発生するとき、❹に移行する。
ARTSによる負荷がARTS対象機体に発生しないとき、❼に移行する。
```
### ❹精度指数計算
```
ARTSによって発生する精度指数を計算する。
計算方法は以下の通り。

(精度指数)=(依存技能の技能行使 Lv)×(精度率)　小数点以下保持
(精度率)=(ARTSに記載の基礎精度率)-[{(有効射程)-(座標差量)}×10%]+(RANGE補正)

(有効射程)-(座標差量)>0のとき、(RANGE補正)=0%
(有効射程)-(座標差量)=0のとき、(RANGE補正)=20%

計算終了後、❺に移行する。
```
### ❺回避指数計算
```
ARTS対象機体は機動の値を参照し、回避指数を発生させることが出来る。
このとき、ARTS対象機体は操縦技能【機動制御】で判定する。
計算方法は以下の通り。

(回避指数)=(【機動制御】の技能行使Lv)×(回避率)　小数点以下保持
(回避率)=(100%)+[10%{(機動)-(精度指数)}]

また、座標が3以上離れた機体の対象となった場合、回避指数を発生させるには操縦技能【危機感知】の技能行使判定を先に行わなければならない。

【危機感知】の技能行使Lv≧(座標差量)のとき、回避指数を発生させることができる。
【危機感知】の技能行使Lv<(座標差量)のとき、回避指数を発生させることができない。(回避指数=0として扱う)

計算終了後、❻に移行する。
```
### ❻ダメージ確定
```
技能使用機体が技能対象機体に与えるダメージを確定し、ダメージ処理を行う。
こうして実際に発生するダメージのことを、発生ダメージと定義する。
計算方法は以下の通り。

(命中指数)=(精度指数)-(回避指数)　小数点以下保持
(発生ダメージ)=(命中指数)-(装甲量)　小数点以下四捨五入

発生ダメージは0以下にはならず、装甲は発生ダメージが最低値となるよう自動的に使用されるものとする。
このとき、装甲の値から使用された装甲量を差し引き、発生ダメージ1につき一つの部位を失う。
失う部位の決定は、以下に従って行う。

1ダメージのとき：被ダメージ機体が決定する。
2~3ダメージのとき：1dxのダイスロールでランダムに決定する。
(x=残りの部位数)
4ダメージ以上のとき：与ダメージ機体が決定する。

処理終了後、❼に移行する。
```
### ❼ARTS使用終了
```
ARTS使用機体はARTSの使用が終了したことを宣言し、②TURN ACTIONを参照して各手順に移行する。
```
```
※小数点以下保持について
浮動小数点誤差の表示を防ぐため、保持する少数は小数点以下第二位までとする。(四捨五入)
```
