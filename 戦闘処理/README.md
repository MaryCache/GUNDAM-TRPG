## 🎯 戦闘の流れ

### 1. 戦闘開始
```
① GMが戦闘開始を宣言
② 【機械操作】でジャミング → 戦闘区域作成
③ /board create で盤面作成
④ 全員が /board add で参加
```

### 2. 1WAVE の流れ
```
SELECT PHASE
  └ 全員が移動先とARTSを秘密裏に予約

ACTION PHASE
  ├ 速度指数で行動順決定
  ├ 優先度ありARTSが先行
  └ 順番にARTS使用処理

END PHASE
  └ 敵が残っていれば次のWAVEへ
```

### 3. ARTS使用処理
```
❶ ARTS使用宣言
❷ 故障判定
❸ ARTS効果発生
❹ 精度指数計算
❺ 回避指数計算
❻ ダメージ確定
❼ ARTS使用終了
```

## 🧮 計算式早見表

### 速度指数
```
速度指数 = 機動制御Lv × 機動
```

### 精度指数
```
精度率 = 基礎精度率 - (射程距離減衰 × 10%) + RANGE補正
精度指数(Raw) = 技能行使Lv × 精度率
最終精度指数 = 精度指数(Raw) × 同調率倍率
```

### 回避指数
```
回避率 = 100% + [10% × (機動 - 精度指数)]
回避指数(Raw) = 機動制御Lv × 回避率
最終回避指数 = 回避指数(Raw) × 同調率倍率
```

### ダメージ
```
命中指数 = 最終精度指数 - 最終回避指数
発生ダメージ = 命中指数 - 装甲 (小数点以下四捨五入)
```
